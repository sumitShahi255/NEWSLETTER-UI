Newsletter Subscription Card

A simple and responsive Newsletter Subscription Card UI built using HTML and Tailwind CSS.
This project demonstrates modern UI design using utility-first CSS with gradients, spacing, and responsive layout.

 Features

 Gradient header design

 Responsive layout using Flexbox

Email subscription form

Privacy-friendly message

Clean and modern UI using Tailwind CSS

Technologies Used

HTML5

Tailwind CSS

CSS (custom stylesheet support)
